CRUD.STORE – E-commerce Frontend Template

CRUD.STORE es una plantilla base de e-commerce desarrollada con HTML5 y CSS3, orientada a la creación de tiendas online modernas. El proyecto se centra en la estructura visual y organización de los componentes principales de una tienda virtual.


>>>> Características

* Estructura HTML semántica

* Diseño moderno para tienda de ropa

* Secciones de productos, marcas y opiniones

* Barra de búsqueda y navegación principal

* Formulario de suscripción

* Código claro y fácil de modificar


>>>> Tecnologías utilizadas

* HTML5

* CSS3

* Tipografías externas (Lilita One, Satoshi, Ubuntu)

* SVG Icons


>>>> Estructura del proyecto

* index.html

* style_index.css

* imgs/

* README.md

>>>> Uso

Abrir el archivo index.html en el navegador y modificar textos, imágenes y estilos según las necesidades del proyecto.
Esta plantilla puede usarse como base para proyectos académicos, portafolio o integración futura con JavaScript y backend.
